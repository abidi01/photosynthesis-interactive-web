## 📖 项目简介

本项目是一个完全基于HTML5、CSS3和JavaScript开发的交互式教学网页，无需安装任何软件，双击HTML文件即可在浏览器中运行。项目包含8个独立的实验模拟页面，涵盖了光合作用研究的经典实验和核心概念。

### ✨ 主要特点

- 🎯 **纯前端实现**：无需服务器，本地即可运行
- 🖱️ **交互式学习**：拖放操作、实时反馈、动画演示
- 📊 **数据可视化**：实时图表展示实验数据变化
- 📱 **响应式设计**：支持电脑、平板、手机多端访问
- 🎨 **精美界面**：现代化UI设计，绿色主题贴合生物学特色
- 🎬 **视频教学**：内置教学视频，辅助理解复杂概念

## 🧪 实验模块

### 1. 希尔实验模拟
- **科学家**：罗伯特·希尔（Robert Hill, 1937-1939）
- **核心发现**：证明光反应可以独立于暗反应进行
- **实验特色**：模拟离体叶绿体在光照下释放氧气的过程

### 2. 阿尔农实验模拟
- **科学家**：丹尼尔·阿尔农（Daniel Arnon, 1954）
- **核心发现**：发现光合磷酸化，证明叶绿体能合成ATP和NADPH
- **实验特色**：展示光能转化为化学能的完整过程

### 3. 贾格道夫实验模拟
- **科学家**：安德烈·贾格道夫（Andre Jagendorf, 1966）
- **核心发现**：验证化学渗透假说，证明质子梯度驱动ATP合成
- **实验特色**：演示酸-碱转换实验，在黑暗中合成ATP

### 4. 卡尔文循环实验模拟
- **科学家**：梅尔文·卡尔文（Melvin Calvin, 1946-1953）
- **核心发现**：揭示CO₂固定和还原的完整途径
- **实验特色**：展示暗反应的三个阶段和物质循环

### 5. 光反应模型
- **内容**：光系统I、光系统II、电子传递链
- **特色**：拖放式填空，理解光反应的详细机制

### 6. 阿尔农2实验模拟
- **内容**：深入理解能量转化过程
- **特色**：详细展示光能如何转化为化学能（有机物）

### 7. 光反应和暗反应之间的联系
- **内容**：ATP、NADPH等物质的循环
- **特色**：拖动模型理解两个阶段的物质交换

### 8. 对接高考
- **内容**：光合作用知识点专项测试
- **特色**：涵盖所有实验内容，检验学习成果

## 📚 使用说明

### 教师使用指南

1. **课堂演示**：
   - 使用投影仪展示实验过程
   - 实时调整参数，观察结果变化
   - 结合科学史讲解实验背景

2. **学生自主学习**：
   - 分发项目文件夹给学生
   - 学生在自己的电脑上探索
   - 完成"对接高考"模块的测试

3. **作业布置**：
   - 要求学生完成特定实验
   - 记录实验数据和观察结果
   - 撰写实验报告

### 学生使用指南

1. **从主页开始**：打开 `main.html`，浏览所有实验模块
2. **按顺序学习**：建议按照希尔→阿尔农→贾格道夫→卡尔文的顺序
3. **动手操作**：尝试拖放、点击、调整参数
4. **观看视频**：遇到难点时观看教学视频
5. **完成测试**：最后完成"对接高考"模块
