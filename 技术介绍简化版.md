## 🎯 项目概述

这是一个使用**HTML、CSS、JavaScript**三种网页技术开发的交互式教学工具，包含8个独立的实验页面，通过统一的导航系统连接在一起。

## 📱 核心技术介绍

### 1. 网页三大技术

**HTML（超文本标记语言）**
- 作用：定义网页的结构和内容
- 比喻：就像房子的框架和布局
- 本项目中：定义了导航栏、实验区域、按钮、视频播放器等元素

**CSS（层叠样式表）**
- 作用：控制网页的外观和样式
- 比喻：就像房子的装修和美化
- 本项目中：设置了绿色主题、圆角按钮、动画效果、响应式布局
- 文件格式：`.css` 文件

**JavaScript（脚本语言）**
- 作用：实现网页的交互功能
- 比喻：就像房子的电器和智能系统
- 本项目中：实现了拖放功能、实验模拟、数据图表、视频控制
- 文件格式：`.js` 文件

## 🔗 页面跳转技术详解

### 技术名称：超链接导航（Hyperlink Navigation）

### 实现原理

**1. HTML超链接标签**
```html
<a href="希尔实验模拟工具.html">希尔实验</a>
```
- `<a>` 标签：HTML中的超链接标签
- `href` 属性：指定跳转的目标文件
- 点击后：浏览器加载目标HTML文件

**2. 统一导航栏设计**
- 所有8个页面都包含相同的导航栏代码
- 导航栏位于页面顶部，固定显示
- 包含8个链接，分别指向8个不同的HTML文件

**3. 相对路径引用**
```
当前文件：main.html
目标文件：希尔实验模拟工具.html
路径写法：href="希尔实验模拟工具.html"
```
- 因为所有HTML文件在同一文件夹，直接写文件名即可
- 不需要写完整路径，浏览器自动在当前文件夹查找

**4. 当前页面高亮显示**
```javascript
// JavaScript代码自动识别当前页面
const currentPage = window.location.pathname.split('/').pop();
// 给当前页面的导航链接添加高亮样式
```
- 使用JavaScript获取当前文件名
- 自动给对应的导航链接添加不同颜色

### 跳转流程

```
用户点击"希尔实验" 
    ↓
浏览器读取href属性（希尔实验模拟工具.html）
    ↓
浏览器加载新的HTML文件
    ↓
新页面显示，导航栏保持不变
    ↓
新页面的导航链接自动高亮
```

## 🎨 主要交互功能的技术实现

### 1. 拖放功能（Drag and Drop）

**技术名称**：HTML5 Drag and Drop API

**实现步骤**：
```javascript
// 1. 设置元素可拖动
<img draggable="true" id="light" src="图片/光.png">

// 2. 监听拖动开始事件
element.addEventListener('dragstart', function(e) {
    // 记录被拖动的元素ID
    e.dataTransfer.setData('text', e.target.id);
});

// 3. 监听放置区域事件
dropZone.addEventListener('drop', function(e) {
    // 获取被拖动的元素ID
    const id = e.dataTransfer.getData('text');
    // 验证是否正确，然后放置
});
```

**关键技术点**：
- `draggable` 属性：使元素可拖动
- `dragstart` 事件：拖动开始时触发
- `dragover` 事件：拖动经过时触发
- `drop` 事件：放下时触发
- `dataTransfer` 对象：传递拖动数据

### 2. CSS动画效果

**技术名称**：CSS3 Animation 和 Transition

**气泡上升动画**：
```css
@keyframes rise {
    0% {
        bottom: 0;
        opacity: 1;
    }
    100% {
        bottom: 300px;
        opacity: 0;
    }
}

.bubble {
    animation: rise 2s linear;
}
```

**关键技术点**：
- `@keyframes`：定义动画的关键帧
- `animation` 属性：应用动画效果
- `transition` 属性：平滑过渡效果

### 3. 实时数据图表

**技术名称**：Chart.js 图表库

**实现代码**：
```javascript
// 1. 创建图表
const chart = new Chart(ctx, {
    type: 'line',  // 折线图
    data: {
        labels: ['0s', '1s', '2s'],  // X轴标签
        datasets: [{
            label: '氧气生成量',
            data: [0, 10, 20]  // Y轴数据
        }]
    }
});

// 2. 更新图表数据
chart.data.labels.push('3s');
chart.data.datasets[0].data.push(30);
chart.update();  // 刷新图表
```

**关键技术点**：
- Chart.js：第三方JavaScript图表库
- 实时更新：使用 `setInterval` 定时器每秒更新
- 数据绑定：JavaScript数组与图表数据同步

### 4. 视频播放控制

**技术名称**：HTML5 Video API

**实现代码**：
```html
<!-- HTML视频标签 -->
<video id="videoPlayer" src="视频/光反应.MP4" controls></video>
```

```javascript
// JavaScript控制
const video = document.getElementById('videoPlayer');

// 播放
video.play();

// 暂停
video.pause();

// 重置到开头
video.currentTime = 0;
```

**关键技术点**：
- `<video>` 标签：HTML5原生视频播放器
- `controls` 属性：显示播放控制条
- JavaScript API：通过代码控制播放、暂停、进度

### 资源引用技术

**相对路径引用**：
```html
<!-- 引用图片 -->
<img src="图片/光.png" alt="光">

<!-- 引用视频 -->
<video src="视频/光反应.MP4"></video>

<!-- 引用CSS -->
```

**路径解析规则**：

- 浏览器自动根据当前HTML文件的位置解析路径
- 所有资源必须保持相对位置不变

### 开场介绍（2分钟）

"同学们，今天我们使用的这个工具是用**网页技术**开发的。网页技术是互联网的基础，我们平时浏览的所有网站都是用这些技术做的。

这个工具使用了三种核心技术：
1. **HTML**：定义网页的结构，比如按钮、文字、图片放在哪里

3. **JavaScript**：实现交互功能，比如拖动、点击、数据计算"

### 页面跳转演示（2分钟）

"现在我演示一下页面之间是怎么跳转的：

（指向导航栏）看到顶部这个绿色的导航栏了吗？这是用HTML的**超链接技术**实现的。

（点击链接）当我点击'希尔实验'这个链接时，浏览器会加载一个新的HTML文件，叫做'希尔实验模拟工具.html'。

（再次点击）点击'光反应模型'，又加载了另一个文件'光反应.html'。

这就像在文件夹里打开不同的Word文档，每个实验都是一个独立的HTML文件，通过导航栏的**超链接**连接在一起。

技术术语叫做**相对路径引用**，因为所有文件在同一个文件夹，所以可以直接通过文件名跳转。"

### 拖放功能演示（2分钟）

"现在看这个拖动功能，这是用**HTML5的拖放API**实现的：

（演示拖动）我拖动这个'光'的图标，这个动作触发了一个叫**dragstart**的事件。

（拖到目标位置）当我把它拖到这个位置上方时，触发了**dragover**事件，告诉浏览器这里可以放置。

（松开鼠标）松开鼠标时，触发**drop**事件，JavaScript代码会验证是否放对了位置。

（显示结果）如果正确，就把图标固定在这里；如果错误，就显示提示信息。

这整个过程都是通过**JavaScript事件监听**实现的。"

### 动画效果演示（1分钟）

"看到这些气泡从下往上冒吗？这是用**CSS3动画**实现的。

我们用CSS定义了一个叫**@keyframes**的动画规则，告诉浏览器：
- 开始时气泡在底部，透明度100%
- 结束时气泡在顶部，透明度0%
- 中间的每一帧浏览器自动计算

这样就产生了平滑的上升和淡出效果。"

"这个实时更新的图表是用**Chart.js图表库**实现的：

（点击开始实验）当实验开始时，JavaScript代码每秒钟：

2. 把新数据添加到数组中
3. 调用图表的**update方法**刷新显示

这就是**数据可视化**技术，把数字变成直观的图表。"

## ❓ 技术相关问题解答

### Q1: 为什么用网页技术而不是做成软件？
**A**: 网页技术的优势：
- 跨平台：Windows、Mac、手机都能用
- 无需安装：双击HTML文件就能打开
- 易于分享：复制文件夹即可
- 易于修改：用记事本就能编辑HTML代码

### Q2: 页面跳转会丢失之前的数据吗？
**A**: 会的。每个页面是独立的，跳转后会重新加载。如果需要保存数据，可以使用：
- localStorage（浏览器本地存储）
- sessionStorage（会话存储）
- Cookie（小型数据存储）

### Q3: 为什么有些功能需要网络？
**A**: 因为使用了CDN（内容分发网络）资源：
- Tailwind CSS：从cdn.tailwindcss.com加载
- Font Awesome：从cdn.jsdelivr.net加载
- 优点：减小文件大小，加载速度快
- 缺点：需要网络连接

### Q4: 可以在手机上使用吗？
**A**: 可以！这叫**响应式设计**：
- 使用CSS媒体查询（@media）
- 根据屏幕大小自动调整布局
- 手机上导航栏会变成竖向排列
- 按钮和文字会自动缩放

### Q5: 如果想修改内容怎么办？
**A**: 可以直接编辑HTML文件：
- 用记事本或VS Code打开HTML文件
- 找到要修改的文字或图片路径
- 保存后刷新浏览器即可看到效果
- 建议先备份原文件
